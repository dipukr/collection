Object-oriented Programming
Object-oriented programming (OOP) is a programming paradigm that organizes code around objects, which are self-contained units that bundle data and functions together. This approach uses concepts like classes (templates for objects), encapsulation (bundling data and methods), inheritance (reusing properties from other classes), and polymorphism (objects taking on many forms) to create modular, reusable, and maintainable code.

An object oriented program is basically a collection of objects and relationships among them and they communicate or interact with each other using message passing. Four main concept of OOPs are abstraction, encapsulation, inheritance and polymorphism which help us to write modular, reusable, extensible and generic software system
